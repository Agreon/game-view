# GameWatch

Overview of game release dates, prices and news

![Screenshot](./screenshot.png)

## Features

- Nightly update of Game data
- Custom Tags
- (E-Mail) Notifications for
    - Price reduction
    - Release date change
    - Release of game
    - Addition to store
    - Addition of Metacritic rating
